# NESTJS X BACKEND

## Description

Online store backend written with NestJS and TypeScript + PostgreSQL.

## Running the app

```bash
# Installation
$ yarn install

# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
